	Извлечение значения ключа name из POST-запроса с {"name": "Mememe"}.

	Проверить релизацию данного задания можно с помощью CURL:

	Например, curl -d "name=yourName" -X POST https://apxpress.herokuapp.com/name
	
    
Маршруты для приложения-калькулятора:

	Add: /add/12/3
	Subtract: /subtract/12/3
	Multiply: /multiply/12/3
	Divide: /multiply/12/3
	Pow: /pow/12/3
	Kramer: /kramer/2/3/4/1/2/3


    Например add/5/6 вернёт 11, а pow/2/3 вернет 8 и т.д.

    Решение системы уравнений Крамера вернёт JSON вида {"x":13,"y":47}
